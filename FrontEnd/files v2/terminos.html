Términos y Condiciones

Condiciones generales de uso de la plataforma Cyntia SIEM

Última actualización: 1 de enero de 2025

1. Información general

Estos Términos y Condiciones regulan el acceso y uso de la plataforma Cyntia SIEM (en adelante, "la Plataforma"), desarrollada como Trabajo de Final de Grado. Al acceder o utilizar la Plataforma, el usuario acepta quedar vinculado por estos términos.

La Plataforma es un sistema SIEM (Security Information and Event Management) de código abierto diseñado para pequeñas y medianas empresas (PYMEs).

2. Definiciones

  • Plataforma: El software Cyntia SIEM, incluyendo su interfaz web, agentes, APIs y documentación.
  • Usuario: Cualquier persona física o jurídica que acceda o utilice la Plataforma.
  • Agente: Software ligero instalado en los equipos del usuario para la recolección de eventos de seguridad.
  • Datos de seguridad: Logs, eventos, alertas y cualquier información procesada por la Plataforma.

3. Condiciones de uso

3.1 Requisitos del usuario

Para utilizar la Plataforma, el usuario debe ser mayor de edad legal y tener capacidad jurídica para aceptar estos términos. En caso de uso empresarial, el usuario declara tener autorización de la empresa para contratar estos servicios.

3.2 Registro y cuenta

Ciertas funcionalidades de la Plataforma requieren la creación de una cuenta de usuario. El usuario se compromete a proporcionar información veraz y actualizada, y a mantener la confidencialidad de sus credenciales de acceso.

3.3 Uso aceptable

El usuario se compromete a utilizar la Plataforma exclusivamente para fines de monitorización y seguridad legítimos. Queda prohibido:

  • Utilizar la Plataforma para actividades ilegales o no autorizadas.
  • Intentar acceder a datos de otros usuarios sin autorización.
  • Realizar ingeniería inversa del software salvo en lo permitido por la licencia MIT.
  • Utilizar la Plataforma para realizar ataques informáticos o pruebas de penetración no autorizadas.
  • Distribuir malware o código malicioso a través de la Plataforma.

4. Licencia de software

Cyntia Core se distribuye bajo licencia MIT, lo que permite su uso, copia, modificación y distribución libre, siempre que se incluya la nota de copyright original.

Las versiones Pro y Enterprise pueden incluir componentes adicionales con licencias comerciales diferentes. Los términos específicos de cada plan se detallan en la sección de Servicios.

5. Protección de datos

El tratamiento de datos personales se rige por nuestra Política de Privacidad, que forma parte integrante de estos Términos.

La Plataforma procesa datos de seguridad (logs, eventos, alertas) de la infraestructura del usuario. Estos datos se tratan conforme al Reglamento General de Protección de Datos (RGPD) y la Ley Orgánica de Protección de Datos (LOPD-GDD).

6. Disponibilidad del servicio

La disponibilidad de la Plataforma depende del plan contratado. Cyntia no garantiza un tiempo de actividad del 100%, si bien los planes Enterprise incluyen SLA con compromisos específicos de disponibilidad.

Cyntia se reserva el derecho de realizar tareas de mantenimiento que puedan afectar temporalmente al servicio, notificando al usuario con la mayor antelación posible.

7. Limitación de responsabilidad

La Plataforma se proporciona "tal cual" (AS IS). Cyntia no garantiza que el software detecte todas las amenazas de seguridad ni que esté libre de errores. El usuario es el único responsable de la seguridad de su infraestructura.

El usuario conserva todos los derechos sobre sus datos de seguridad procesados por la Plataforma.
